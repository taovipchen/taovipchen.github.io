Linux 常用高频的命令

Linux 常用高频的命令

2019-10-30 7 min read

磁盘管理 cd

Linux cd命令用于切换当前工作目录至 dirName(目录参数)。

进入当前目录命令
cd .
切换当前目录的上级目录
cd ..
切换当前目录的上两级目录
cd ../..
切换当前用户的home目录
cd ~
切换到/home/Taochen目录
cd /home/Taochen

磁盘管理 mkdir

Linux mkdir命令用于建立目录。

新建目录

在home目录下新建Taochen目录。

mkdir /home/Taochen
新建多个目录

在home目录下新建Taochen、iTaochen目录。

mkdir /home/Taochen /home/iTaochen
新建多层级目录

在/home/Taochen目录下新建/tao/chen目录。

mkdir -p /home/Taochen/tao/chen

磁盘管理 ls

Linux ls命令用于显示指定工作目录下之内容.

列出当前目录中所有的子目录和文件
ls
列出目录下的所有文件包含隐藏文件
ls -a
列出文件的详细信息（包含文件型态、权限、拥有者、文件大小）
ls -l
列出当前目录中所有.log文件的详细内容
ls -l *.log

文件管理 mv

Linux mv命令用来为文件或目录改名、或将文件或目录移入其它位置。

移动文件夹

若/home/iTestOps目标目录已存在，将源目录/home/Taochen移动到目标目录/home/iTtaochen；

若目标目录/home/iTaochen不存在则修改目录名为 /home/iTaochen。

mv /home/Taochen /home/iTaochen
移动文件

将tomcat.log文件移动到目标目录/home/Taochen 。

mv tomcat.log

移动tomcat.log文件到上级目录。

mv tomcat.log ../
移动文件并重命名

当前目录tomcat.log 移动到/home/Taochen/ 目录，并重命名为tomcat_back.log。

mv tomcat.log /home/Taochen/toamcat_back.log

文件管理 cp

Linux cp命令主要用于复制文件或目录。

复制文件

复制tomcat.log文件 到 /home/iTaochen目录。

cp tomcat.log /home/iTaochen
复制文件夹

复制文件夹 Taochen 到目录 /home/iTaochen

cp -r Taochen  /home/iTaochen

文件管理 scp

Linux scp命令用于Linux之间复制文件和目录。

从 本地 复制到 远程

将本地/home/iTaochen/目录的tomcat.log 文件复制到远程172.119.23.75服务器的/home/TestOps 目录。

scp .home/iTaochen/tomcat.log
root@172.119.23.75: /home/Taochen
从 远程 复制到 本地

将远程172.119.23.75服务器的/home/Taochen目录下的tomcat.log 文件 复制到本地/home/iTaochen/目录中。

scp root@172.119.23.75: /home/Taochen/tomcat.log
/home/iTaochen/tomcat.log

文件管理 find

Linux find命令用来在指定目录下查找文件。

根据文件名查找文件

在/home/iTaochen/ 目录及其子目录下面查找名字为tomcat.log的文件 。

find /home/iTaochen/ -name tomcat.log

在/home/iTaochen/目录及其子目录中查找任何扩展名为“.log”的文件。

find /home/iTaochen/ -name *.log

根据更改时间查找文件

在/home/iTaochen/目录下查找更改时间在3日以内的文件

find /home/iTaochen/ -mtime -3

在/home/iTaochen/目录下查找更改时间在1日以外的文件

find /home/iTaochen/ -mtime +1
将当前目录其其下子目录中所有一般文件列出
find . -type f
根据文件内容查找文件

在/home/iTaochen/目录下，查找文件内容中含有Taochen的所有文件。

find /home/iTaochen/ -type f | xargs grep "Taochen"
根据更改时间删除文件

在/home/iTaochen/目录下，删除两天之内没有修改过的文件

find /home/iTaochen/ -mtime +2 -print | xargs rm -rf

文件管理 rm

Linux rm命令用于删除一个文件或者目录。

删除文件

删除/home/iTaochen/目录下的文件tomcat.log

rm /home/iTaochen/tomcat.log
强行删除文件

强制删除/home/iTaochen/目录下的文件tomcat.log。

rm -rf /home/iTaochen/tomcat.log

删除/home/iTaochen/目录下的所有.log文件。

rm -rf /home/iTaochen/*.log
删除目录

删除/home目录下的 iTaochen文件夹

rm -r /home/iTaochen/

强行删除/home目录下的 iTaochen文件夹，慎用。

rm -rf /home/iTaochen/
删除全部文件和目录

删除/home/iTaochen/目录下的所有内容。

rm -rf /home/iTaochen/*

文件打包压缩 tar

Linux tar命令用于打包文件。

打包单文件为.tar文件

压缩当前目录中tomcat.log 到log.tar 中。

tar -cvf log.tar tomcat.log
压缩文件为.tar.gz文件

将当前目录，所有.log文件压缩成Taochen_bak.tar.gz。

tar -zcvf Taochen_bak.tar.gz *.log
打包多文件

将前目录中以.log结尾的所有文件打包到all.tar 中。

tar -cvf  all.tar *.log
打包目录

打包/home/Taochen/目录到当前目录Taochen_bak.tar中。

tar -cvf Taochen_bak.tar /home/Taochen/
解压.tar文件

解压Taochen_bak.tar到当前目录。

tar -xvf Taochen_bak.tar
解压.tar.gz文件

解压Taochen_bak.tar.gz 到当前目录。

tar -zxvf Taochen_bak.tar.gz

备注：

  • .tar只是进行打包，并没有压缩。 可用 tar-cv f进行打包，用 tar-xvf 进行解包。

  • .tar.gz是既打包又压缩 。tar –czvf [压缩包名] [文件或目录]打包并且压缩，tar –zxvf [压缩包名]解包并且压缩。注意：-z表示bzip的操作！

  • .gz 只压缩。只能针对文件。如果需要对一个目录下的全部文件压缩，则用gzip -r [目录]。gzip [文件名]进行压缩，gzip –d [文件名]进行解压。

系统管理 ps

Linux ps命令用于显示当前进程的状态。

显示进程信息

ps -A

查看python进程

ps -ef | grep python

显示所有进程信息，连同命令行

ps -ef

wget

Linux系统中的wget是一个下载文件的工具。

下载文件

下载testfile.zip文件

wget http://www.taochen-tc.com/testfile.zip

下载文件并重命名

下载testfile.zip文件，并存储名为TestOps.zip

wget -o taochen.zip http://www.taochen-tc.com/testfile.zip

后台下载文件

下载testfile.zip文件，后台形式下载

wget -b http://www.taochen-tc.com/testfile.zip
